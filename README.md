# ADXRay Game Spy

输入任意游戏名，自动从 ADXRay 提取广告投放数据。

不需要安装 Python，不需要大模型，下载 exe 双击即用。

## 功能

- 搜索游戏在 ADXRay 的投放数据
- 提取：投放概览（素材数/计划数/投放周期）
- 提取：渠道分布（投放媒体/广告位）
- 提取：热门文案 Top 20
- 提取：投放趋势
- 输出文本报告

## 使用方式

### 方式一：直接下载 exe（推荐）

1. 下载 `adxray-spy.exe`
2. 双击运行
3. 首次使用：输入游戏名 → 弹窗浏览器 → 登录 ADXRay
4. 之后：输入游戏名 → 点击"开始提取" → 出报告

### 方式二：源码运行

```bash
pip install -r requirements.txt
python adxray_spy_gui.py      # 图形界面
python adxray_spy_core.py 原神 # 命令行
```

## 前置条件

- ADXRay 账号（付费版）
- Windows 系统
